# Github Archive Program?
* 깃허브에서 핵전쟁, 자연재해 등으로 인해 **인류의 컴퓨팅 자원의 유실되는 것을 막기 위해** 데이터를 보관하는 프로젝트이다.
  * 보관소는 노르웨이령 스발바르제도 폐광 내 깊이 250m지하에 위치한다고 한다.
  * 여담으로, 전 세계 100만 개 씨앗을 보관 중인 '스발바르 국제 종자 저장고(SGSV)'도 이곳에 위치해있다.
